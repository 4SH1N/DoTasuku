# DoTasuku

DoTasuku is a simple Todo application built as a learning project to strengthen my Ruby on Rails skills through hands-on practice.

The goal of this project is not to build a production-ready application, but to apply and reinforce concepts I've learned while exploring modern Rails development.

## Objectives

- Practice building CRUD functionality with Ruby on Rails.
- Gain practical experience with Turbo Frames and Turbo Streams.
- Improve my understanding of Tailwind CSS for responsive UI design.
- Learn how different Rails components work together in a real application.
- Develop cleaner project structure and better coding practices.

## Tech Stack

- Ruby on Rails
- Hotwire (Turbo Frames & Turbo Streams)
- Tailwind CSS
- SQLite (development)

## Status

This project is actively developed as I continue learning Rails and experimenting with new features.
